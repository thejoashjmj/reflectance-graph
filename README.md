# Additional info to note:

## **trialForCheckingReflectanceGraph folder contains the Arduino IDE .ino file**

## BACKEND SETUP 
1. Clone the repo
2. Open terminal in folder and run :
   ```bash
   npm install
   ```

## Arduino IDE SETUP
1. Open the .ino file in the Arduino IDE.
2. Update the ssid and password for your WiFi.
3. Change serverUrl to your laptop's IP address.
4. Upload to ESP32.


## Running server 
1. To start the server, run in terminal of folder:
  ```bash
  node server.js
  ```
2. Then, to view graph,  open browser and go to
   ```bash
   http://localhost:3000
   ```
